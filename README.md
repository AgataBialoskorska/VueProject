## How to:

### Clone the project to a selected directory on your computer.
```
git clone https://github.com/AgataBialoskorska/VueProject.git
```

### Go to the project directory and install it.
```
npm i
```

### Run development server

```
npm run dev
```

## Other Commands: 

### Run cypress test

- development server: 
```
npm run e2e-run
```
- production server:
```
npm run e2e-prod
```
